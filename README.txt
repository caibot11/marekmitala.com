# My Personal Website

This repository contains the source code for my personal website built with Next.js and TypeScript. The website features a distinctive cyberpunk aesthetic with custom styling, animations, and a futuristic design inspired by the gritty vibe of Cyberpunk 2077.

## Features

- **Cyberpunk Design:** A dark, neon-infused aesthetic with accents of yellow and cyan.
- **Responsive Layout:** Optimized for both desktop and mobile devices.
- **Modular Components:** Built with React components and CSS Modules for clear separation of concerns.
- **Light/Dark Mode:** Theme toggle to switch between light and dark modes.
- **Contact Form & Social Links:** Consistent styling across the website for a seamless user experience.
